## üì± iPhone / Apple Calendar (One-Tap)

1.  Open **Settings**
2.  Go to **Calendar ‚Üí Calendar Accounts**
3.  Tap **Add Account ‚Üí Add Other ACcount...**
4.  Tap **Subscribed Calendar**
5.  Paste the HTTPS URL above in the **Server** field
6.  Tap **Next ‚Üí Save**

------------------------------------------------------------------------

## ü§ñ Android (Google Calendar)

1.  Open https://calendar.google.com
2.  Click **+** next to *Other calendars*
3.  Select **From URL**
4.  Paste the HTTPS URL above
5.  Click **Add calendar**

It will sync automatically to Android devices using that Google account.

------------------------------------------------------------------------

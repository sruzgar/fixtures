## 📱 iPhone (Apple Calendar)

1.  Open **Settings**
2.  Go to **Calendar → Calendar Accounts**
3.  Tap **Add Account → Add Other ACcount...**
4.  Tap **Subscribed Calendar**
5.  Paste the HTTPS URL above in the **Server** field
6.  Tap **Next → Save**

------------------------------------------------------------------------

## 🤖 Android (Google Calendar)

1.  Open https://calendar.google.com
2.  Click **+** next to **Other calendars**
3.  Select **From URL**
4.  Paste the HTTPS URL above
5.  Click **Add calendar**

It will sync automatically to Android devices using that Google account.

------------------------------------------------------------------------

## 🐛 Troubleshooting

If updates don't appear:

-   iPhone: Toggle the subscription off/on
-   Google Calendar: Refresh browser or allow time for sync
